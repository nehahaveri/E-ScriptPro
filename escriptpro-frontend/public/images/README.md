# 3D Landing Page — Image Setup

Save your 3D medical images to `public/images/` with these exact filenames:

| Filename                  | Which image to use                                         |
|---------------------------|------------------------------------------------------------|
| `hero-medical.png`        | Medical box with stethoscope, pills, heart, syringe (5th)  |
| `clipboard-check.png`     | Blue clipboard with medical cross icon (1st)                |
| `prescription-pen.png`    | Clipboard with pen writing on it (2nd)                      |
| `medical-search.png`      | Clipboard with magnifying glass and icons (4th)             |
| `checklist.png`           | Orange checklist with blue checkmarks and pencil (3rd)      |
| `hospital-bed.png`        | 3D hospital bed with IV drip (6th)                          |

All images should be **PNG with transparent background** for best results.
Recommended size: **800×800px** or larger.
